Class Compiler

A compiler is a class handling the creation of source code for a specific platform, providing a function to export it as a string. The download of the file should be handled separately in the src/components/App/menu.tsx.

This file provides an example on how to retrieve data from the store, how to process it, and how to assemble it, in this case to produce an Arduino Scratch file, that can be compiled and uploaded to a board.

Utilizes template code contained in the blueprint/ folder.
